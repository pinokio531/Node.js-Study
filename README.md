# Node.js-Study
생활코딩 강의를 통한 node.js 스터디 진행

## Node.js 시작하기(Terminal or CMD)
  - node.js 사이트에서 npm과 node 다운로드 진행
  - $ node --version
      -> 버전이 출력되면 정상 설치된 것임
      
  - $ mkdir (code-name)
      -> (code-name) 디렉토리 생성
      
  - Editor를 이용하여 신나게 코딩
      -> ex)atom, sublimeText
  
  - $ node (code-name)
      -> 코드 실행
      
  - 실행되고 있는 서버 종료시
      -> ctrl + c
